    • Defined in src/sudoKeyArchive/sudoKeyArchive.ts:78

    Decrypts and unarchives the keys in this archive.

    throws

    KeyArchiveDecodingError If the archive is unable to be decoding according to the required structure.

    throws

    KeyArchiveIncorrectPasswordError If the archive is a secure archive and the password is unable to be used to decrypt the encrypted keys.

    throws

    KeyArchivePasswordRequiredError If the archive is a secure archive but no password is provided.

    throws

    KeyArchiveNoPasswordRequiredError If the archive is an insecure archive but a password is provided.

    Parameters

    • password: undefined | ArrayBuffer

      The password to use to decrypt the archive if its a secure archive otherwise must be undefined.

    Returns Promise<void>
